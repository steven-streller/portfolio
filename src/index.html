  • Masterprojekt: Metaheuristische Optimierung (Whale Optimization Algorithm)

    Im Rahmen eines Masterprojekts wurde der naturinspirierte Whale Optimization Algorithm (WOA) analysiert, erweitert und zur Lösung kombinatorischer Optimierungsprobleme eingesetzt.

    Der Algorithmus wurde für diskrete Problemstellungen adaptiert und auf das Traveling Salesman Problem (TSP) sowie das Sequential Ordering Problem (SOP) angewendet. Hierfür wurden bestehende mathematische Modelle weiterentwickelt und algorithmisch umgesetzt.

    Zur Verbesserung von Laufzeit und Ergebnisqualität wurden zusätzliche Optimierungsansätze wie eine Greedy-Heuristik sowie eine dynamische Anpassung von Iterations- und Populationsparametern konzipiert, implementiert und evaluiert. Paper

    Metaheuristiken Optimierungsalgorithmen Java TSP / SOP Algorithmische Analyse

    Container Security Docker Open Container Initiative (OCI)
